<h2><a id="user-content-modules" class="anchor" href="#modules" aria-hidden="true"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Modules</h2>

<p>Modules can be installed by downloading or git cloning into the emoncms/Modules folder. Be sure to check for database updates in Administration menu after installing new modules:</p>

<ul>
<li><p><a href="https://github.com/emoncms/dashboard">Dashboards module</a>, required for creating, viewing and publishing dashboards: </p></li>
<li><p><a href="https://github.com/emoncms/app.git">App Module</a> - Application specific dashboards e.g. MyElectric, MySolar</p></li>
<li><p><a href="https://github.com/emoncms/config.git">Config</a> - In-browser emonhub.conf editor and emonhub.log log viewer. git clone</p></li>
<li><p><a href="https://github.com/emoncms/wifi.git">Wifi Module</a> - <a href="https://guide.openenergymonitor.org/setup/connect/">Wifi configuration interface designed for use on the emonPi</a></p></li>
</ul>

<p>There are many other modules such as the event module and openbem (open source building energy modelling module) that are available, check out the <a href="https://github.com/emoncms">Emoncms repo list</a></p>
